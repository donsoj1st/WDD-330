math object function

  1. Math.abs() method returns the absolute value of a number
  2. Math.ceil() method will round a number up to the next integer, or remain the same if it is already an integer
  3. Math.floor() method will round a numberdownto the next integer, or remain the same if it is already an integer
  4. Math.round() method will round a number to the nearest integer
  5. Math.trunc() method that returns the integer-part of a number – that is, it gets truncated at the decimal point
  6. Math.exp() method will raise a number to the power of Euler’s constant
  7. Math.pow() method will raise number of the first argument to the power of the number of second argument
  8. Math.sqrt() method returns the positive square root of a number
  9. Math.cbrt() method was introduced in ES6, which returns the cube root of numbers
  10. Math.max() method returns the maximum number from its arguments
  11. Math.min() method unsurprisingly returns the minimum number from the given arguments

Event

  1. addEventListener() method is called on a node object, the node to which the event listener is being applied
  2. event can be add to the body of documnet using document.body.addEventListener('click',doSomething);
  3. screenX and screenY properties show the number of pixels from the left and top of the screen respectively where the event took place
  4. clientX and clientY properties show the number of pixels from the left and top of the client that is being used
  5. pageX and pageY properties show the number of pixels from the left and top, respectively, where the event took place in the document
  6. You should be very cautious of attaching both a click and doubleclick event to the same element
  7. mousedown and mouseup events
  8. there are three events that occur when a user presses a keyboard which are : keydown , keypress and keyup
  9. touchstart event occurs when a user initially touches the surface
  10. touchend event occurs when a user stops touching the surface
  11. touchmove event occurs after a user has touched the screen then moves around without leaving
  12. touchenter event occurs when a user has already started touching the surface, but then passes over the element to which the event listener is attached
  13. touchleave event occurs when the user is still touching the surface, but leaves the element to which the event listener is attached

Document Object Model - DOM

  • All nodes have a numerical code to signify what type they are
  • Node lists are array-like objects, but they are not arrays
  • getElementById() - reference to the element with a unique id attribute
  • getElementsByTagName() - return a live node list of all the elements with the tag name that is provided as an argument
  • getElementsByClassName() - live node list of all elements that have the class name that is supplied as an argument
  • document.querySelector() - use CSS notation to find thefirstelement in the document that matches that matches a CSS selector provided as an argument
  • document.querySelectorAll() - uses CSS notation but returns a node list ofallthe elements in the document that match the CSS query selector
  • jQuery is a popular JavaScript framework that makes it very easy to find elements on a page using a CSS-style syntax
  • childNodes property is a list of all the nodes that are children of the node concerned
  • children property only returns anyelementnodes that are children of that node
  • firstChild property returns the first child of a node
  • lastChild property returns the last child of a node
  • parentNode property returns the parent node of an element
  • nextSibling property returns the next adjacent node of the same parent
  • previousSibling property returns the previous adjacent node
  • createElement() method that takes a tag name as a parameter and returns that element
