array, set and loops

  1. An array is an ordered list of values
  2. Destructuring an array is the process of taking a value out of an array and representing them as individual value
  3. The length property can be used as part of the index to find the last item in an array
  4. pop() -used to remove the last item from an array
  5. shift()- used to remove the first item from an array
  6. push() - used to appends a new item to the end of an array
  7. unshift() - used to appends a new item to the beginning of an array
  8. concat() - used to merge one or more arrays together
  9. join() - used to turn array into a string that comprises all items in the array, separated by commas
  10. slice() - creates a subarray, chopping a slice of the original array
  11. splice() - removes items from an array and inserts new items in their place
  12. reverse() - permanently reverses the order of the array
  13. sort() - sort the order of the array
  14. multidimensional array - array of arrays
  15. Sets: ata structure that represents a collection of unique values, so it cannot include any duplicate values
  16. Maps are new in ES6 and are ordered lists of key-value pairs
  17. We can use an if and else statement to control the flow of code
  18. The switch statement can be used instead of multiple if and else statements
  19. A while loop and do ... while loop can be used to repeat a block of code while a condition is still true
  20. A for loop works in a similar way to a while loop, but has a different syntax
  21. A for-of loop can be used to iterate over an array
  22. Sets and maps are enumerable, so thet can be looped over using a for-of loop

Function

  1. To define a function literal we can use a function declaration
  2. Another way of defining a function literal is to create a function expression
  3. A function can also be declared using the constructor Function()
  4. If you have assigned a function to a variable, you need to place parentheses after the variable to invoke it as a function
  5. All functions return a value, which can be specified using the return statement, which comes after the return keyword
  6. any parameters a function needs are set when the function isdefined
  7. When a function isinvoked, it is provided with arguments
  8. Arrow functions can be identified by the 'arrow' symbol, => that gives them their name
  9. Arrow functions are always anonymous, so if you want to refer to them, you must assign them to a variable
  10. callback - A function that is passed as an argument to another
  11. forEach() - loop through the array and invoke a callback function using each value as an argument
  12. map() - it returns a new array that replaces each value with the return value of the callback function
  13. reduce() - cumulatively combines each result to return just a single value
  14. filter() - returns a new array that only contains items from the original array that return true when passed to the callback

Arrow functions have some advantages over other ways of declaring functions

  • They are much less verbose than normal function declarations
  • Single parameters don't need putting into parentheses
  • The body of the function doesn't need placing inside a block if it's only one line
  • The return keyword isn't required if the return statement is the only statement in the body of the function
  • They don't bind their own value of this to the function
